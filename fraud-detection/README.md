# Credit Card Fraud Detection

Machine learning project for detecting fraudulent bank account applications on a highly imbalanced dataset.

* 1M samples, 32 features, fraud rate = 1.1%
* Performed EDA, preprocessing, and feature engineering
* Evaluated Logistic Regression, QDA, Random Forest, and XGBoost
* Best model: XGBoost (AUC = 0.953, TPR = 0.79)

**Tech:** Python, Pandas, scikit-learn, XGBoost
